# digiskill_AI_for_Python
My assignments and projects for the DigiSkills  course AI-for-Python. This repository tracks my progress as I complete various exercises throughout the program.

## What is in this project?
1. **Tax Calculation:** This task calculates tax based on a salary input. It uses `if`, `elif`, and `else` statements to decide which tax rate to apply.
2. **Word Dictionary:** This task takes a list of words and counts how many letters are in each word. It uses a `for` loop and a `dictionary` to store the results.
3. **Chat System:** This is a small chat app made with OOP (Object Oriented Programming). It uses `classes` like User, Message, and ChatRoom to manage users and chat history.

## How to use them?
- You can run these files using any Python editor or the terminal.
- For the Tax task, just enter your salary when asked, and it will show the result.
- For the Word task, it will automatically print the dictionary of word lengths.
- For the Chat System, it runs a test script showing users joining and sending messages.

## Concepts used
- Conditional statements (if/else)
- Loops and Dictionaries
- Object-Oriented Programming (Classes and Methods)
